# Генератор учебных заданий

Streamlit приложение для автоматической генерации учебных заданий на основе компетенций и индикаторов с использованием встроенных шаблонов.

## Описание

Приложение автоматически генерирует учебные задания различных типов на основе шаблонов:
- Задания закрытого типа на установление соответствия
- Задания закрытого типа на установление последовательности
- Задания комбинированного типа с выбором одного правильного ответа
- Задания комбинированного типа с выбором нескольких правильных ответов
- Задания открытого типа с развернутым ответом

## Требования

- Python 3.8+
- Установленные зависимости из requirements.txt

## Установка

1. Клонируйте репозиторий или скачайте файлы

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

```bash
streamlit run app.py
```

Приложение откроется в браузере по адресу `http://localhost:8501`

## Использование

### Шаг 1: Подготовка файлов

Вам понадобятся два файла:

1. **Excel файл с данными** (например, `megaphops.xlsx`)
   - Столбец A: Код и наименование компетенции
   - Столбец B: Индикатор достижения компетенции
   - Столбец C: Дисциплина / модуль / практика
   - Столбец D: Уровень сложности (тип задания)
   - Столбец E: Задание (будет заполнено автоматически)
   - Столбец F: Ключ (ответ) (будет заполнено автоматически)

2. **Файл с промптами** (например, `promt`)
   - Содержит инструкции для генерации каждого типа задания
   - Формат: тип задания + TAB + инструкция

### Шаг 2: Использование приложения

1. Загрузите файл megaphops.xlsx в интерфейс приложения
2. Файл promt загружается автоматически из репозитория
3. Выберите диапазон строк для обработки (начальная и конечная строка)
4. Нажмите кнопку "Сгенерировать задания"
5. Дождитесь завершения генерации
6. Просмотрите результаты в таблице

### Шаг 3: Экспорт результатов

1. После генерации нажмите кнопку "Скачать результат (Excel)"
2. Сохраните файл с заполненными столбцами "Задание" (E) и "Ключ" (F)

## Особенности

- Генерация заданий на основе встроенных шаблонов (без использования AI API)
- Автоматическое определение типа задания из столбца D
- Генерация заданий с учетом компетенций и индикаторов
- Прогресс бар для отслеживания процесса генерации
- Предпросмотр данных перед обработкой
- Экспорт результатов в Excel
- Обработка ошибок и пропуск строк с некорректными данными
- Быстрая работа без необходимости подключения к внешним API
- Бесплатное использование

## Решение проблем

### Ошибка "ModuleNotFoundError"
Убедитесь, что установили все зависимости:
```bash
pip install -r requirements.txt
```

### Некорректная генерация
- Убедитесь, что файл `promt` содержит правильный формат данных
- Проверьте, что столбец D содержит корректные типы заданий
- Задания генерируются на основе шаблонов, поэтому могут требовать ручной доработки
